# 🌍 Chernobyl Simulator [![License](https://img.shields.io/github/license/UrbanLevente/Chernobyl-Simulator.svg)](https://github.com/UrbanLevente/Chernobyl-Simulator/blob/main/LICENSE)

An indie **3D game** with a goal of simulating the Chernobyl Nuclear Power Plant (NPP).

---

## 🛠️ Building Instructions

Follow these simple steps to get **Chernobyl Simulator** up and running:

> 1. Clone the repository:
>    ```sh
>    git clone https://github.com/UrbanLevente/Chernobyl-Simulator.git
>    ```
> 2. Check out the `main` branch:
>    ```sh
>    git checkout main
>    ```
> 3. Open the `Sim.sln` solution file in [Visual Studio](https://visualstudio.microsoft.com/downloads/ "Visual Studio").
> 4. In the **Solution Configuration**, select `Dist`.
> 5. Build the project by pressing `Ctrl + Shift + B`.
